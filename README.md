# AV5-TPII

## Primeira execução

### Backend

```bash
npm i
npm run start:completo
```

### Frontend

```bash
npm i

npm run build
npm run preview
```

Ou, para desenvolvimento:

```bash
npm run dev
```

Após iniciar o frontend, abra o link exibido no terminal no navegador.

---

## Execuções seguintes

### Backend

```bash
npm run run
```

### Frontend

```bash
npm run dev
```

ou

```bash
npm run preview
```
